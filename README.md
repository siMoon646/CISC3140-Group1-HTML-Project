# TEAM HTML PROJECT

Objective: Practice collaborative Git workflow while learning HTML basics.

Instructions:
1. Team Lead creates a new GitHub repository
2. Add all team members as collaborators
3. Each team member creates their own HTML file (yourname.html)
4. Your HTML page should include:
   - Proper HTML5 structure (<!DOCTYPE html>, <html>, <head>, <body>)
   - A heading with your name
   - A short bio paragraph about yourself
   - An unordered list of 3 hobbies or interests
   - At least one image (can be a placeholder)
   - A link to your favorite website
5. Push your file to the shared repository
6. Create an index.html that links to all team members pages

Git Requirements:
- Each member must make at least 2 commits
- Use meaningful commit messages
- Practice pulling before pushing
